# 使用说明

## 首次启动

1. 运行程序。
2. 在首次设置窗口输入并确认解锁密码。
3. 保存后进入主界面，同时程序驻留系统托盘。

## 常用操作

- 点击主界面的“立即遮罩”。
- 或使用托盘菜单“立即遮罩”。
- 或按 `Ctrl + Alt + L`。

## 解锁

1. 在主显示器遮罩界面输入密码。
2. 点击“解锁”。
3. 若连续输错，会触发短暂延迟限制。

## 设置

可配置：

- 开机自启
- 启动后立即遮罩
- 空闲自动遮罩
- 快捷键启用
- 多显示器覆盖
- 时钟与遮罩文案
- 修改密码

## 忘记密码

该软件不提供后门解锁。忘记密码时：

1. 退出程序。
2. 删除 `%AppData%\DesktopPrivacyShield\config.json`。
3. 重新启动程序并重新设置密码。
